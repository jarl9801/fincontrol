# Nothing Design System — DEPRECATED

This skill is no longer active. FinControl migrated to the **NEXUS.OS** design system on 2026-04-23.

→ Read `nexus-design.md` instead.

The `nd-*` / `n-*` utility classes (`nd-label`, `nd-mono`, `nd-display`, etc.) still render through compatibility shims in `src/index.css`, but they now produce NEXUS.OS styling. New code should prefer the canonical Nexus equivalents (`label-mono`, `font-mono`, `font-display` + `font-light`).
